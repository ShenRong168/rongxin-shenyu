# 榮心紳語專案總索引

最後更新：2026-06-29

這份文件是專案的共同上下文入口。它只記錄跨分支共識、決策、待辦與索引，不作為聊天逐字稿。每次進行總討論前，請先閱讀本文件，再視需要追讀「分支摘要」與「相關檔案位置」中列出的文件。

## 專案目標

- 建立「榮心紳語」的試營運網站與營運流程。
- 服務定位為心態整理、溝通陪跑與行動釐清，並明確區分於心理諮商、心理治療、醫療、診斷或危機介入。
- 以靜態網站先行發布，逐步補齊表單、付款、錄音、個資、交易條款與開站前檢核。

## 重要決策

- 網站採靜態頁面：主要內容在 `index.html`，樣式在 `styles.css`。
- 公開網址規劃為 GitHub Pages：`https://shenrong168.github.io/rongxin-shenyu/`。
- 試營運單次服務 60 分鐘，採隨緣付費新台幣 100 至 1200 元。
- 試營運付款方式先使用 LINE 轉帳，不宣稱為 LINE Pay 合作商店。
- 線上服務平台採 Google Meet。
- 服務對象僅接受年滿 18 歲者，且服務進行當下須位於台灣。
- 不錄影；錄音採使用者個別同意，並於服務開始前再次確認。
- 錄音保存期限為服務後 7 日，保存於 Google Drive「榮心紳語」資料夾，並以下載連結交付。
- 正式聯絡管道目前使用 `anchen918@gmail.com`。
- 網站必須清楚揭露服務界線、危機資源、個資告知、付款與退款規則。
- 行銷主軸從「溫柔陪伴」收斂為「把說不清的內耗，整理成能面對的問題與下一步」。
- 內容經營主平台先採 Threads，網站文章作為信任沉澱與轉換頁。
- 內容語氣採「溫柔 + 一點刺痛」：開頭抓痛點，中段接住，結尾導向小練習或前置盤點。
- `PROJECT_CONTEXT.md` 作為總索引；分支討論細節寫入 `docs/context/`。
